This is an unofficial documentary page of uMod.org web API

Please keep in mind that it only contains known endpoints and might be incomplete, include deprecated and removed entries

Assets

Manifest

Contains information about all the published packages, game data and other things used by uMod.Agent
assets.umod.org/umod.manifest.json

Community

Contributors

Returns list of all community contributors
umod.org/community/contributors.json

Plugins

Search

Returns list of published plugins

Query parameters

  • page
  • tags
umod.org/plugins/search.json

Plugin info

Information about specific plugin
umod.org/plugins/npc-drop-gun.json

Plugin versions

Information about published versions of specified plugin

Query parameters

  • page
umod.org/plugins/npc-drop-gun/versions.json

Docs

Hooks

Returns information about documented hooks
umod.org/documentation/hooks/rust.json
